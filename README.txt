Веб-приложение "Форум Питонистов"
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Для запуска приложения надо запустить файл "main.py".

При запуске приложения на навигационной панели есть кнопки для авторизации и регистрации, 
отправляющие на другие страницы с формами для авторизации и регистрации соответственно.
При входе на главной странице можно добавлять, удалять и изменять новости.
После авторизации можно добавлять и удалять комментарии под каждой новостью.
Нажатие на кнопку бренда навигационной панели открывает главную страницу.

База данных в которой хранятся данные о новостях и пользователях хранится в файле "db/forum.sqlite".
Пароли хранятся в зашифрованном виде. Все классы для работы основной программы находятся в папке "data".
Изображения находятся в папке "static/img". А все файлы разметки страницы и шаблоны находятся в папке "templates".
